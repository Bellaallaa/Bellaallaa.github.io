具体步骤

    Node.js是一个基于 Chrome V8 引擎 的 JavaScript 运行时环境，它可以让你在服务器端运行 JavaScript，而不仅仅是在浏览器中。Node.js 是一个 事件驱动、非阻塞式 I/O 模型，适合开发高性能、可扩展的网络应用程序，特别是处理并发请求时。

    如何确认是否安装了nodejs?

常用命令

命令 作用
hexo init 初始化一个新的博客
hexo new "My New Post" 新建一篇博客
hexo n 同上
hexo clean 清理缓存文件与已生成的静态文件(常用语hexo g 前)
hexo generate 生成静态网页(在public/文件夹中)
hexo g 同上
hexo server 启动本地服务器, 默认端口4000, 可预览博客
hexo s 同上
hexo deploy 部署网站到远程
hexo d 同上
hexo list [type] 列出某类内容(如posts, pages, tags 等)
hexo version 查看hexo的版本号
hexo help 产看命令帮助文档

一些注意…

  1. 在部署时, 有时会因为网络问题出现报错, 可以过段时间再次尝试
  2. 使用hexo d命令部署后需要等待一段时间才能在网页看到更新后的显示, 可以使用ping github.com进行测试
