🧩 一、什么是 tmux？

tmux（Terminal Multiplexer） 是一个终端复用器，可以：

  • 在一个终端里开多个“窗口”；
  • 让程序在后台持续运行；
  • 即使关闭 VSCode 或 SSH 连接，也能再回来继续。

💡 四、进阶：VSCode + tmux 自动化体验更好

你可以安装以下插件：

  1. “tmux” 插件（在 VSCode 扩展市场里搜 tmux）；
  2. 或者 “Remote - WSL” 插件（如果你在 Windows 上用 WSL）。

这些插件可以：

  • 在 VSCode 的终端面板中直接显示多个 tmux 窗口；
  • 支持快捷键在 tmux 窗格间切换；
  • 自动保存 tmux 会话布局。

✅ 五、常用快捷键速查表

动作 按键
创建新会话 tmux new -s name
分离会话（后台运行） Ctrl+B, 然后按 D
列出所有会话 tmux ls
重新连接会话 tmux attach -t name
关闭会话 tmux kill-session -t name
在会话中开新窗格 Ctrl+B, 然后按 %（垂直）或 "（水平）
在窗格间切换 Ctrl+B, 然后用方向键
